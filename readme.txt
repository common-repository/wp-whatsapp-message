Chat Message Button (Antes WhatsApp Message) es un plugin de WordPress que sirve para agregar un botón en una página a través de un shortcode y permite a los usuarios enviar un mensaje de texto directamente a número de WhatsApp.

== Description ==
Chat Message Button (Antes WhatsApp Message) es un plugin de WordPress que sirve para agregar un botón en una página a través de un shortcode y permite a los usuarios enviar un mensaje de texto directamente a número de WhatsApp.

== Frequently Asked Questions ==

= Lista de shortcodes disponibles en plugin =
Chat Message Button, ¿Como usar el shortcode?

// Configruación por defecto (cambiar número: código de área + teléfono).
[cmb-message phone=&quot;0000000000&quot;] 

// Agrgando una clase para cambiar los estilos CSS.
[cmb-message phone=&quot;0000000000&quot; class=&quot;nombre de la clase&quot;] 

// Cambiando la imagen por defecto.
[cmb-message phone=&quot;0000000000&quot; icon=&quot;url de la imagen&quot;] 

// Cambiando el texto del mensaje.
[cmb-message phone=&quot;0000000000&quot; usr_text=&quot;lorem ipsum dolor&quot;] 

// Cambiando el texto del enlace por defecto.
[cmb-message phone=&quot;0000000000&quot; text=&quot;lorem ipsum dolor&quot;] 

= Chat Message Button, Otras funciones: =
[cmb-message class=&quot; &quot;] // Deshabilitar solo los estilos CSS.

= ¿Más funciones? =
Por ahora solo he creado estos shortcodes, en futuras versiones desarrollaré más funciones para que disfutres de este plugin.

== Installation ==
Instalación desde el repositorio oficial de WordPress:
1. Escribe Daniel Esparza en la barra de búsqueda.
2. Selecciona el plugin para instalar.
3. Activa el plugin.
4. Lee las instrucciones de uso en la sección '<> DE Plugins' dentro del menú de administración de WordPress.
5. ¡Disfruta!

Instalación manual (con archivo zip):
1. Escribe Daniel Esparza en la barra de búsqueda del repositorio oficial de WordPress.
2. Descarga el plugin.
3. Sube el plugin desde el panel de administración de WordPress, Plugins -> Añadir nuevo -> Subir plugin -> Seleccionar archivo (buscar el archivo en nuestro equipo) -> Instalar ahora.
4. Activa el plugin desde el menú 'Plugins' de WordPress.
5. Lee las instrucciones de uso en la sección '<> DE Plugins' dentro del menú de administración de WordPress.
6. ¡Disfruta!
